# WorkTime 

環境：  
- min sdk 25  
- File > Settings > Build > Instant Run > 取消勾選 `Enable Instant Run ...`

## 功能
- 紀錄排班、加班、請假資訊
- 紀錄班表、時薪、時數
- 計算應得薪資
- [*待新增的功能*](https://hackmd.io/@rgbkoi/rJFUDnxyr)  

## 首頁
<img src="https://github.com/alexxss/WorkTime/blob/master/screenshots/homepage.png" width="300"/>

- 顯示**本月應得薪資**
- 顯示金額=固定排班+加班-請假
- 簡列當月**加班所得**&**請假扣除**
- 可查看**詳細資訊**/**清空資料**
